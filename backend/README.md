# Easygenerator Backend

## Installation 
```shell
yarn install
```

## Compile and run the project
- **Run the development server**
    ```shell
    yarn run start
    ```
- **Run with the watch mode**
    ```shell
    yarn run start:dev
    ```
- **Run with the production mode**
    ```shell
    yarn run start:prod
    ```

## Run tests
- **Run unit tests**
    ```shell
    yarn run test
    ```
- **Run e2e tests**
    ```shell
    yarn run test:e2e
    ```
- **Run test coverage**
    ```shell
    yarn run test:cov
    ```
