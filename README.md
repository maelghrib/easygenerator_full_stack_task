# Easygenerator Full Stack Task
A production-ready authentication system that allows a user to sign up and sign in to the application.

## Tech Stack
- TypeScript
- Nestjs
- MongoDB
- React.js
- Next.js

## Getting Started

### Run the Backend
```shell
cd backend
yarn run start
```
Then visit http://localhost:3333

### Run the Frontend
```shell
cd frontend
yarn dev
```
Then visit http://localhost:3000
